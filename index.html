What is the Web?
The World Wide Web includes all web pages accessible via the internet. To design a website, we need to learn three essential languages: HTML, JavaScript, CSS.

HTML
Structure of an HTML Page
The <html> element contains the <head> and <body> elements. We will later learn about each element and where it is located in an HTML page.

HTML Basics
HTML pages consist of several elements that include two parts: an opening tag (<tag>) and a closing tag (</tag>). Note that the closing tag starts with "/" before the element name.

Types of Elements in HTML
The <br> element is used like the Enter key in texts.
The <pre> element is used to display content with the same spacing.
The <p> element is used to write a paragraph.

Horizontal Line
To create a horizontal line, we use the <hr /> element.

Headings
Headings range from <h1> to <h6>, where the size depends on the number. The higher the number, the smaller the text.

Images
We use the <img> element to add an image, specifying the source (src) and the image title (alt).

Links
We use the <a> element to add a link. The ID attribute can be used to specify target labels.

Lists
Unordered Lists
We use the <ul> element for unordered items with the <li> element for each item.
Ordered Lists
We use the <ol> element for ordered items with the <li> element for each item.

Tables
We use the <table> element to define a table with the <tr> element for rows, <th> for table headers, and <td> for table data. A table cell can contain all types of HTML elements.

HTML Forms
Forms are used to collect data from the user and send it to a specific server for processing. All form elements must be written inside the form content.

Form Elements
We use the <input>, <label>, <select>, <textarea>, <button>, and <option> elements to specify different types of inputs in the form.

##CSS SECTION

Introduction to CSS
CSS stands for Cascading Style Sheets. It is used to style and layout web pages. CSS can change color, font, spacing, and layout of HTML elements.

How CSS Works
CSS can be added to HTML in three ways: Inline CSS: Directly in the HTML element using the style attribute. Internal CSS: Inside a <style> tag in the HTML document's <head> section. External CSS: In a separate .css file linked to the HTML document.

Inline CSS
Suitable for small changes, not recommended for larger projects.

Internal CSS
Good for styling single-page websites.

External CSS
Best for larger projects with multiple pages.

CSS Syntax
A CSS rule consists of a selector and a declaration block. The selector targets the HTML element. The declaration block contains declarations separated by semicolons. Each declaration consists of a property and a value.

Selectors
Element Selector: Targets HTML elements by their tag name. Class Selector: Targets elements with a specific class. ID Selector: Targets an element with a specific ID.

Colors and Backgrounds
Color Property: Changes text color. Background-Color Property: Changes background color. Can use color names, hex codes, or RGB values.

Fonts
Font-Family Property: Changes the font. Font-Size Property: Changes the size of the font.

Text Alignment
Text-Align Property: Aligns text. Values: left, right, center, justify.

Margins and Padding
Margin: Space outside an element. Padding: Space inside an element. Can set different values for top, right, bottom, and left.

Borders
Border Property: Adds a border around an element. Can customize width, style, and color.

Box Model
Elements are boxes with content, padding, border, and margin. Visual representation helps understand layout and spacing.

Display Property
Display Property: Controls the layout of an element. Common values: block, inline, inline-block, none.

Positioning
Position Property: Controls the position of an element. Values: static, relative, absolute, fixed.

CSS Flexbox
Flexible layout system for arranging elements.

CSS Grid
Powerful layout system for creating complex designs.



##JS

JavaScript Basics
Introduction to JavaScript
JavaScript is a programming language used to make web pages interactive. It can change HTML and CSS and respond to user actions.

Adding JavaScript to HTML
Use the <script> tag to add JavaScript to an HTML file.

Basic Syntax
Statements end with a semicolon. Comments can be single-line or multi-line. Variables can be declared using var, let, or const.

Data Types
Common data types include strings, numbers, booleans, and arrays.

If Statements
If syntax includes if, else if, and else.

Basic Operations
Operations include arithmetic, assignment, comparison, and logical operations.

Functions
A function is a block of code designed to perform a task.

Name & ID
Naming an HTML object allows for easy reference, while giving an ID to an HTML object makes it uniquely identifiable. These are important for DOM manipulation.

JavaScript DOM Manipulation and Form Handling
Document Object Model (DOM)
The DOM represents the HTML document as a tree structure. JavaScript can access and change elements in the DOM.

Selecting Elements
Use document.getElementById to select an element by its ID. Use document.querySelector to select elements using CSS selectors.

Changing Content
Use innerHTML to change the content of an element.

Changing Styles
Use the style property to change CSS styles.

Event Listeners
Add event listeners to respond to user actions.

Handling Forms
Get form input values and handle form submission.

Creating Elements
Create a new element and append it to the DOM.

Removing Elements
Remove an element from the DOM.
